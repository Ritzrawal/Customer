# Manage-Customer
**_`Clone the github project`_**
```
$ https://github.com/Ritzrawal/Customer.git
```
**_`Navigate to project directory`_**
```
 $cd Customer
 $cd client
```
```
 $cd Customer
 $cd server
```

**_`Install the dependencies`_**
```
 
 $ npm install 

```


**_`Start the Application on web-server`_**
```
 $cd Server
 $ node server.js 
 $ cd client
 $ npm start
```
 #### **Simple build for production** ####

```
 $ npm run build 
```
